# To-Do List & Reminders Application


A lightweight, intuitive desktop application built in Python using the Tkinter GUI toolkit. This application allows users to manage their daily tasks efficiently by adding, deleting, and clearing tasks in real-time.


## Features

1.Add Tasks: Quickly type and insert new tasks or reminders into your list.

2.Delete Tasks: Remove specific, completed, or unwanted tasks by selecting them.

3.Clear All Tasks: Wipe the entire list clean with a single click for a fresh start.


## UI Concepts Used

1.Listbox: To display and manage the collection of tasks dynamically.

2.Buttons: Interactive widgets to trigger the Add, Delete, and Clear actions.

3.Lists (Backend): Python lists handle data storage and state management behind the scenes.


## Prerequisites

Before running the application, ensure you have Python installed on your system:

Python 3.x


Note: Tkinter comes pre-installed with standard Python installations on Windows and macOS.
